## Overview

Q-ARK is a browser extension designed to help developers and QA engineers generate reliable element locators for web automation testing. We are committed to protecting your privacy and being transparent about our data practices.

## Information We Collect

### Data We DO NOT Collect
- **Personal Information**: We do not collect names, email addresses, or any personal identifiers
- **Browsing History**: We do not track or store your browsing history
- **Website Content**: We do not collect or store content from websites you visit
- **User Analytics**: We do not use analytics services or tracking pixels
- **Cookies**: We do not set or read cookies for tracking purposes

### Data We DO Collect (Locally Only)
- **Extension Settings**: User preferences and configuration settings stored locally in your browser
- **Generated Locators**: Temporarily stored in browser memory during active use only
- **Usage Patterns**: Basic functionality usage stored locally for feature improvement

## How We Use Information

### Local Processing Only
- All element analysis and locator generation happens locally in your browser
- No data is transmitted to external servers
- No data is shared with third parties
- No data is used for advertising or marketing

### Local Storage
- Extension settings are stored using Chrome's local storage API
- Data remains on your device and is not synchronized across devices
- You can clear this data by removing the extension

## Data Sharing and Third Parties

**We do not share any data with third parties.** Q-ARK operates entirely offline and does not communicate with external servers.

## Permissions Explained

Q-ARK requests the following permissions:

### activeTab
- **Purpose**: Access the currently active tab to analyze DOM elements
- **Usage**: Only when you explicitly activate the element picker
- **Scope**: Limited to the tab you're currently viewing

### scripting
- **Purpose**: Inject content scripts to analyze page elements
- **Usage**: Only when you use the element picker or evaluation features
- **Scope**: Limited to active tab during extension use

### storage
- **Purpose**: Save your extension preferences locally
- **Usage**: Store settings like preferred code generation language
- **Scope**: Local browser storage only, never transmitted

## Data Security

- **Local Processing**: All operations happen within your browser
- **No Network Requests**: Extension does not make external network calls
- **Secure Storage**: Uses Chrome's secure local storage APIs
- **No Data Transmission**: Zero data leaves your device

## Children's Privacy

Q-ARK does not knowingly collect information from children under 13. The extension is designed for professional developers and QA engineers.

## Changes to Privacy Policy

## Data Retention

- **Session Data**: Cleared when you close the browser or disable the extension

## Your Rights

You have the right to:

- **Delete**: Remove all data by uninstalling the extension
